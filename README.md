# Vite + Vue3 + Pinia + 官网实战 

## 项目介绍

本项目是一个官网开发项目，用于展示和推广在线教育课程。主要功能包括登录、课程视频播放、支付以及课程管理等模块，采用了 Vite + Vue3 + Pinia 技术栈进行开发。

## 功能模块

#### 登录功能

账号密码登录
短信验证码登录

登录滑块验证
登录验证和导航守卫

#### 课程视频播放

防盗链机制

内网穿透

#### 支付功能

微信支付
支付宝支付

#### 其他功能

课程资料下载
课程分类和排序处理物车功能
性能优化等...

## 运行

  ```shell
# 1. 进入拷贝目录下安装依赖
npm i 
# 2. 运行
npm run dev
  ```
